# Retail Sales Paginated Report

## Project Overview
This project demonstrates a professional Paginated Report built using Power BI Report Builder.

The report analyzes retail sales data and presents key metrics and detailed transaction records.

## Tools Used
- Power BI Report Builder
- Excel
- GitHub

## Features
- KPI indicators (Total Sales, Total Profit)
- Detailed sales table
- Exportable PDF report
- Paginated layout for printing

## Dataset
1000 rows of retail sales data including:

- Order ID
- Date
- Region
- Product
- Category
- Customer
- Sales
- Quantity
- Profit
